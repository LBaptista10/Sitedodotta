SITE DOTTA — NOVO VISUAL

ARQUIVOS
- index.html
- logo-dotta.png

COMO PUBLICAR
1. Extraia o ZIP.
2. No repositório Sitedodotta do GitHub, substitua o index.html antigo por este.
3. Envie também logo-dotta.png para a raiz do repositório.
4. Confirme o commit. A Vercel deve atualizar automaticamente.

O QUE JÁ FUNCIONA
- Layout responsivo para celular.
- Logo centralizada.
- Botões Reservar agora e Serviços.
- Modal com os serviços.
- Seleção Society/Futsal.
- Data, horário, nome, telefone e observações.
- Validação visual do formulário.
- Link Seus lances para o GravaJogo.

IMPORTANTE
Esta versão é um protótipo visual funcional no navegador.
O banco de dados, disponibilidade real dos horários, painel administrativo
e envio definitivo da reserva serão conectados em uma próxima etapa.
